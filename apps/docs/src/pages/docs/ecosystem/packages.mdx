# Ecosystem Packages

The Ecopages ecosystem is composed of several modular packages. You only install what you need.

## Core Packages

| Package | Purpose |
| :--- | :--- |
| **[`@ecopages/core`](https://jsr.io/@ecopages/core)** | The heart of Ecopages. Contains the application adapter, router, and base plugin classes. |
| **[`@ecopages/logger`](https://www.npmjs.com/package/@ecopages/logger)** | Unified logging utility used across the entire ecosystem. |

---

## Integrations

These packages add support for different rendering engines.

| Package | Purpose |
| :--- | :--- |
| **[`@ecopages/kitajs`](https://jsr.io/@ecopages/kitajs)** | Fast JSX rendering using KitaJS. |
| **[`@ecopages/react`](https://jsr.io/@ecopages/react)** | React 19 support with SSR and hydration. |
| **[`@ecopages/lit`](https://jsr.io/@ecopages/lit)** | Lit Web Components integration with SSR support. |
| **[`@ecopages/mdx`](https://jsr.io/@ecopages/mdx)** | Markdown with JSX (React-powered by default). |

---

## Processors & Loaders

Tools for processing assets and extending Bun's loading capabilities.

| Package | Purpose |
| :--- | :--- |
| **[`@ecopages/image-processor`](https://jsr.io/@ecopages/image-processor)** | Build-time image optimization and responsive image generation. |
| **[`@ecopages/postcss-processor`](https://jsr.io/@ecopages/postcss-processor)** | Processes CSS through PostCSS and Tailwind. |
| **[`@ecopages/bun-postcss-loader`](https://jsr.io/@ecopages/bun-postcss-loader)** | Allows importing CSS directly as strings in JS/TS. |
| **[`@ecopages/bun-inline-css-plugin`](https://jsr.io/@ecopages/bun-inline-css-plugin)** | Inlines CSS directly into the JS bundle for components. |

---

## Utilities

| Package | Purpose |
| :--- | :--- |
| **[`@ecopages/radiant`](https://www.npmjs.com/package/@ecopages/radiant)** | A minimalist web component library for reactive UI. |
| **[`@ecopages/scripts-injector`](https://www.npmjs.com/package/@ecopages/scripts-injector)** | Helper for lazy-loading scripts in the browser. |

---

## Installation Tip

You can install multiple packages at once using Bun:

```bash
bunx jsr add @ecopages/core @ecopages/kitajs @ecopages/image-processor
```
